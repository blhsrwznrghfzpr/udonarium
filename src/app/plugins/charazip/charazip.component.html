○対応システム

  • キャラクターシート倉庫 : {{ info.name }}  
  • キャラクター保管所 : {{ info.name }}  

○使い方

  1. キャラクターのURLを入力する
    (キャラクターシート倉庫であれば https://character-sheets.appspot.com/SYSTEM/edit.html?key=XXXXXX のようなもの)
    (キャラクター保管所であれば https://charasheet.vampire-blood.net/XXXXXXX のようなもの)
  2. キャラコマ生成を押す
  3. ユドナリウム用のキャラデータ(zipファイル)がダウンロードされます
  4. ダウンロードしたzipファイルをユドナリウムにドラッグ＆ドロップ
